# Arduino-VGA-ram
Arduino sketch to to display data from RAM chip on VGA screen
<br><br>
![](https://nop.koindozer.org/gh/vga/ram-vga4.png)

- 2 470Ω resistors
- 2 68Ω resistors
- 2 10KΩ resistors
- 23LCV1024 chip (RAM)
- Potmeter
- 2 buttons

# Dependencies
[VGAX](https://github.com/smaffer/vgax)
